序言(半生科大情 一世核物心)

1960年秋，当我从上海水产学院家中走进位于嘉定南门的上海科技大学的大门时，我不曾想到，我将会在这里学习五年，工作四十二年，如今退休后还住在这里。位于嘉定南门的上海科大，1958年由上海科学分院各院所与学校对应专业共建，1960年是第一年独立招生，我所在的原子能系核物理专业，是与上海原子核所合办的，当时全国正在大办原子能，我们系还有华东地区科委送来代培生，我们核物理专业一个班就有40余名学生，还有工人班同学。学校还在建造中，没有食堂，吃饭要从南门走到城中体育场，大礼堂没有，开学典礼借的是嘉定电影院。就在这里，满怀豪情的千余学子开始了他们向科学进军之旅。

回望这段跨越半个多世纪的岁月，从青涩学子到白发老者，从求学求知到深耕科研，上海科技大学早已融入我的骨血。这片土地承载了我的青春、我的热爱，也见证了我国核物理与核技术应用从无到有、从弱到强的艰难起步与蓬勃发展。那些饥寒交迫却斗志昂扬的求学时光，那些坎坷波折却从未放弃的科研岁月，那些良师益友的相伴相守、家国使命的初心坚守，都化作了我生命中最珍贵的记忆，值得我细细诉说、永久珍藏。

计算物理：我的科研起点

我的科研工作始终围绕计算物理展开。在系统开展蒙特卡罗方法研究之前，早期十年里，我已在凝聚态物理、界面测量、理论分析等方面做了大量半经验模型与数值计算工作，以方程求解、数值计算为主，为后续研究打下了坚实基础。
