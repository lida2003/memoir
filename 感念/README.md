感念·记恩师与同行挚友

一、师恩难忘，风骨长存
冯永祥先生一生坎坷却心怀家国，出狱后全心投入学术交流与人才培养，我的NIEL相关研究也深受他启发。他是我一生的榜样，知遇之恩永难忘。

二、黄金搭档，情谊绵长
罗文芸博士是我多年的重要合作者，专业扎实、严谨负责，在模型构建、数据处理及科研协作中发挥关键作用，是我科研路上最可靠的伙伴。

[李达](https://github.com/lida2003/)虽非专职参与团队，却在程序开发、技术难题攻关中多次提供关键支持，尤其在罗文芸赴美期间，为团队节省大量时间精力，是不可或缺的得力伙伴。

三、同行相助，共筑科研路
我的科研工作离不开国内外同行支持。特别感谢日本大阪大学田畑教授，他的半经验模型与程序资料为我早期研究奠定基础。我们先后引进俄罗斯、西班牙、欧洲、日本等多国程序，消化吸收后完成适配优化。

感谢国家自然科学基金支持，2006年由罗文芸牵头、樊胜合作，我们在引进俄罗斯标准蒙特卡罗质子输运程序SHIELD的基础上，针对中能质子的非电离能量损失（NIEL）开展了深度改造与适配优化，建立了适用于空间辐射环境的NIEL计算方法，相关成果发表于《高能物理与核物理》2006年刊，为半导体材料的辐射效应评估提供了重要支撑。

感谢梁维修先生引荐军工项目启动经费，为早期研究提供重要保障。同时感谢原子能院等单位协作，让研究成果落地应用。

四、半世相伴，深情相守
我的老伴富知愚，是我上海科大1960届同届同学，磁学专业出身。他动手能力强、科技检索与多语种翻译能力突出，常年为我翻译外文资料、协助科研工作，是我背后最坚实的支撑。如今我们携手走过金婚，相濡以沫五十载，是我人生最大的福报。

科研之路漫漫，幸有良师引路、挚友协作、爱人相伴，核物理事业的薪火也在传承中生生不息。
